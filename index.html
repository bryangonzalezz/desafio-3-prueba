Mis Servicios

Grabacion

La etapa de grabación es el momento para plasmar todos los elementos que conforman las ideas musicales de cada proyecto. Este proceso se puede potenciar con un buen trabajo de pre-producción para optimizar los recursos y el tiempo a la hora de grabar.

Mezcla

Una vez grabados todas las partes de cada elemento comienza el proceso de edición y mezcla. Una etapa que combina diferentes procesos técnicos al servicio de la creatividad.

Maztering

Trabajamos para mejorar y lograr un resultado dentro de los standards comerciales en terminos de niveles sonoros. También se busca lograr consistencia en el audio resultante para asegurarnos que se traslade apropiadamente de un sistema de reproducción a otro
